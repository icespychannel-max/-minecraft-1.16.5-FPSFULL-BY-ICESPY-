🚀 FPSFULL by IceSpyTM — Сборка для слабых ПК
Эта сборка создана специально для тех, кто играет на «печках» и старых ноутбуках. Оптимизирована для игры на серверах (например, ReallyWorld).

ВНИМАНИЕ: Сборка предназначена ТОЛЬКО для игры на серверах. В одиночном мире будет 2–4 FPS из-за нагрузки на процессор. На серверах — стабильные 20–40 FPS.

📦 Как установить сборку:
Распаковка: Скопируйте папку со сборкой в директорию instances вашего лаунчера (PollyMC или Prism Launcher).

Настройка Java (ОБЯЗАТЕЛЬНО):

Откройте настройки этого инстанса в лаунчере.

Перейдите в раздел Java.

Нажмите кнопку «Автообнаружение» (Auto-detect).

Выберите путь к вашей установленной Java (рекомендуется Java 8).

Выделение памяти:

В тех же настройках Java установите «Максимальное выделение памяти».

Для ПК с 4 ГБ ОЗУ ставьте 1040 MB или 2040 MB. Больше ставить не нужно — системе тоже нужен воздух!
🎮 Как выжать максимум FPS (Советы от автора):
Режим F11: Всегда играйте в полноэкранном режиме. Это дает огромный прирост плавности, так как Windows перестает тратить ресурсы на рабочий стол.

Прогрузка: После захода на сервер постойте на месте 30–60 секунд, пока прогрузятся чанки и игроки. После этого FPS стабилизируется.

Фоновые процессы: Перед игрой закройте браузер и лишние программы.
